    About this dashboard

    Note: This dashboard is not based on live data, it uses a static metadata import (which is refreshed monthly). The metadata of the organization is accessible for any DataverseNL user, thus it can be checked and audited. More info contact ub-dataverse@maastrichtuniversity.nl

    The current data source considers the Maastricht University DataverseNL instance . More information about the UM Dataverse is available at DataverseNL support, as well as the Terms of Use and Operational Guidelines . More rules or checks can be considered in the future. However only metadata checks that can reasonably be assessed from available Dataverse attributes can be included.

    This dashboard supports curator monitoring and service improvement, in line with the Research Data Management Code of Conduct . It does not replace human review, legal assessment, privacy review, ethical assessment, or disciplinary data stewardship judgement.

    Please visit the My Research Information Profile application to curate your research outputs and datasets.
